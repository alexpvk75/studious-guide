Available VR pages

  • HomeXR — entry point for WebXR VR controls and navigation to all plots.
  • Sin Plot XR — interactive 3D sine curve with VR controller rotation and translation.
  • Tan Plot XR — interactive 3D tangent curve with live controller interaction.
  • Atan Plot XR — interactive 3D arctangent curve with VR transform controls.
  • Cos Plot XR — interactive 3D cosine curve with VR axis controls.
  • Custom Plot XR — interactive 3D f(x) = 2*x - 1/x curve with VR controls.
  • a*x - b/x Plot XR — adjustable coefficients with live formula display and VR interaction.
  • Sphere Plot XR — 3D sphere with VR sliders controlling radius and center x coordinate.
